Jeeves Bot Command Cheat Sheet

Character Management Bot Commands

Command Description Example
/authorize Links your Discord account with your World of Warcraft account for character management. /authorize
/characters set-main Sets your main character for the guild roster. /characters set-main
/characters view Displays your linked characters. /characters view
/characters hide Hides a character from the public roster. /characters hide
/characters ignore Ignores a character so it won't be shown or managed. /characters ignore
/characters sync Syncs your character data with Jeeves. /characters sync

Useful Bot Commands

Command Description Example
/summon @rolename Mentions all users with the specified role. /summon @Raiders
/time Shows the current server time. /time
/affix Displays the current Mythic+ affixes. /affix
/worldboss us/oc Shows the current world boss for the US or Oceanic region. /worldboss us or /worldboss oc
/realm-status Displays the status of your WoW realm. /realm-status
